== README


Project Title:  BLOG
Author:         Muhammad Usman
Email:          muhammad.usman@tkxel.com

Technologies:   Ruby(1.9.3) + Rails(4.0.0) + Device + Paperclip + Twitter Bootstarp

Description:
It is a simple blog application in which I used Device gem for authentication.
Members can create their account and can post statuses and also can comment on their posted status. Members can
view only those posts which are posted by the logged-in member.
